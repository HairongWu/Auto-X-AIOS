# Auto-X AIOS

The Auto-X AIOS is a RTOS system based on NuttX powered by Auto-X Engine. The goal of Auto-X AIOS is to build an intelligent operating system for edge devices (such as PX4, Spresense boards), which intends to embed Auto-X Models into the OS.
